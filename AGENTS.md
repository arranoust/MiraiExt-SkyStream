# MiraiExt-SkyStream Project Guide

## Overview
Repository of SkyStream plugins (extensions) for streaming anime and movies. Each plugin is a standalone JavaScript file (`plugin.js`) with a `plugin.json` manifest.

## Repository Structure
```
MiraiExt-SkyStream/
├── anidb/          # AniDB (anidb.app) - EN anime, Cloudflare-protected
├── animeonsen/     # AnimeOnsen (animeonsen.xyz) - EN anime, API-based
├── animexin/       # AnimeXin - EN donghua
├── anizone/        # AniZone (anizone.to) - EN anime, Livewire-based
├── idlix/          # Idlix - ID movies & series
├── layarkaca/      # LayarKaca - ID movies & series
├── otakudesu/      # OtakuDesu - ID anime
├── samehadaku/     # Samehadaku - ID anime
├── torrentio/      # Torrentio - Torrent streams
├── dist/           # Built/bundled plugins (auto-generated by CI)
├── repo.json       # Repository manifest
└── AGENTS.md       # This file
```

## Plugin Architecture

### Core Functions (4 required)
Every plugin must implement these global functions:

1. **`getHome(cb)`** - Returns dashboard categories
   - Returns: `{ success: true, data: { "CategoryName": [MultimediaItem, ...], ... } }`
   - Special: `"Trending"` → Hero Carousel, other names → Thumbnail Rows

2. **`search(query, cb)`** - Search functionality
   - Returns: `{ success: true, data: [MultimediaItem, ...] }`

3. **`load(url, cb)`** - Fetch anime/movie details + episodes
   - Returns: `{ success: true, data: MultimediaItem }` (with episodes array)

4. **`loadStreams(url, cb)`** - Fetch playable video links
   - Returns: `{ success: true, data: [StreamResult, ...] }`

### Helper Classes (Recommended)
```javascript
new MultimediaItem({
  title, url, posterUrl, type, year, score, duration,
  status, contentRating, description, tags, episodes,
  syncData, trailers, cast, headers
})

new Episode({
  name, url, season, episode, rating, runtime,
  airDate, dubStatus, posterUrl, description, headers
})

new StreamResult({
  url, quality, source, headers, subtitles
})
```

### Type Mapping
- `type: "series"` - TV anime (default for most anime)
- `type: "movie"` - Movies
- `type: "anime"` - OVA, ONA, Special
- `type: "livestream"` - Live content

### dubStatus Values
- `"sub"` - Subtitled (Japanese audio)
- `"dub"` - Dubbed (English audio)
- `"none"` - No audio info

## Key Conventions

### Cloudflare Bypass
Sites with Cloudflare protection (e.g., anidb.app) require:
```javascript
var UA = 'okhttp/4.12.0';
var HTML_HDR = { 'User-Agent': UA, Accept: 'text/html,application/xhtml+xml,*/*;q=0.8' };
```

### Response Handling
```javascript
function getBody(res) {
    if (!res) return '';
    if (typeof res === 'string') return res;
    return typeof res.body === 'string' ? res.body : JSON.stringify(res.body);
}
```

### HTML Parsing
```javascript
async function parseHtml(html, selector, attr) {
    try {
        var raw = await parse_html(html, selector, attr);
        if (!Array.isArray(raw)) return [];
        return raw.map(function (item) {
            if (!item) return '';
            if (typeof item === 'string') return item;
            if (attr === 'text') return item.text || '';
            return item.attr || item[attr] || '';
        });
    } catch (_) { return []; }
}
```

### Parallel HTTP
```javascript
async function httpParallelGet(requests) {
    if (typeof http_parallel === 'function') {
        var res = await http_parallel(requests.map(r => ({
            method: 'GET', url: r.url, headers: r.headers || HTML_HDR
        })));
        return requests.map((r, i) => ({
            body: String((res[i] && (res[i].body || res[i].text)) || ''),
            url: r.url
        }));
    }
    // Fallback to Promise.all
}
```

## Deployment
- Push to `main` branch → GitHub Action auto-bundles plugins
- Version bump required in `plugin.json` for updates
- Deploy command: `skystream deploy -u <URL>` (local)

## Plugin-Specific Notes

### AniDB (`anidb/`)
- Uses `okhttp/4.12.0` User-Agent to bypass Cloudflare
- Type detection from HTML: `<dt>Type</dt><dd><a href="/browse?type=TV">TV</a></dd>`
- Sub/dub detection via `/api/frontend/episode/{id}/languages`
- Episode URL format: `{animeUrl}/{epNumber}?type=sub` or `?type=dub`
- AniZip API for episode metadata: `https://api.ani.zip/mappings?anilist_id={id}`

### AnimeOnsen (`animeonsen/`)
- OAuth2 authentication required
- Search token from homepage meta tag

### AniZone (`anizone/`)
- Livewire-based, requires session initialization
- Dynamic content loading via Livewire updates

### Samehadaku (`samehadaku/`)
- Uses `okhttp/4.12.0` User-Agent
- Multiple embed resolvers (filedon, wibufile, pixeldrain)
